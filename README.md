# REST API Ticket System

This project is a REST API built using Node.js, Express, and PostgreSQL.  
The API allows users to retrieve, create, and update help desk support tickets.

The project was developed as part of a REST API development assignment.

---

## Technologies Used

- Node.js
- Express.js
- PostgreSQL
- pg (Node PostgreSQL client)
- dotenv
- cors

---

## Live API

Deployed on Render:

https://rest-api-ticket-system.onrender.com

Example endpoint:

https://rest-api-ticket-system.onrender.com/api/v1/tickets

---

## API Endpoints

### GET Tickets

Returns all tickets.
